# Fibonacci-clock-pygame-python

clock build with pygame.</br>
Refreshes every 5min.

### Color dictionnary

- RED : hour.
- GREEN : minuts.
- BLUE : both.

### Square value

- small : 1
- smedium : 2
- medium : 3
- big : 5

### calculation
   
   (RED/ BLUE) square value*5 : (GREEN/ BLUE) square value*5
